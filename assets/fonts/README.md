# Poster fonts

Place these font files here for poster PDF generation:

- **GeneralSans-Bold.ttf** — heading/title (23pt). Also: .otf, .woff
- **IvyPrestoDisplay-Regular.ttf** — names (20pt). Also: IvyPrestoDisplay-Light.ttf, .otf, .woff

If the files are missing, the poster generator falls back to Helvetica.
